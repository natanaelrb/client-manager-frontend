# 📋 Client Manager Frontend

O **Client Manager** é uma interface desenvolvida para gerenciar o fluxo de clientes de uma aplicação. Este projeto atua como a camada de apresentação para uma **API REST desenvolvida em Java**, demonstrando a integração Full Stack e o consumo de serviços.

> **Status do Projeto:** 🚧 Em Desenvolvimento (Estrutura Inicial)

---

## 🚀 Tecnologias que serão utilizadas

O projeto utiliza tecnologias modernas para garantir uma interface rápida e responsiva:

* **React** + **Vite**: Ambiente de desenvolvimento de alta performance.
* **JavaScript (ES6+)**: Lógica de componentes e integração.
* **HTML5 & CSS3**: Estrutura e estilização.
* **Integração**: Consumo de API REST via Fetch/Axios.
* **Axios / Fetch API:** Comunicação assíncrona com o backend Java.

---

## 🎯 Objetivo e Funcionalidades

O objetivo principal deste repositório é servir como portfólio para demonstrar o domínio da comunicação entre um frontend React e um ecossistema **Java Backend**, com foco principal na prática da integração entre Frontend e Backend (Java/Spring Boot), simulando cenários reais de uma aplicação de gestão corporativa.

### 🔌 Funcionalidades planejas:
- [ ] **Listar clientes:** Recuperação de dados do banco via API.
- [ ] **Cadastrar clientes:** Envio de novos registros para o servidor.
- [ ] **Editar clientes:** Atualização de informações existentes.
- [ ] **Remover clientes:** Exclusão lógica/física de registros.
- [ ] **Listagem Dinâmica:** Visualização de todos os clientes registrados via API.
- [ ] **Fluxo de Cadastro:** Formulário validado para inserção de novos clientes.
- [ ] **Edição de Dados:** Atualização rápida de informações existentes.
- [ ] **Remoção Segura:** Exclusão de registros com confirmação de ação.
- [ ] **Feedback ao Usuário:** Notificações de sucesso/erro durante as requisições.

# ⚙️ Instalação e Execução
 **Pré-requisitos**
- **Node.js** (versão LTS recomendada)
- **Gerenciador de pacotes** (NPM ou Yarn)

## 📦 Instalação e Execução

Para rodar o projeto localmente, siga os passos abaixo:

1. **Clone o repositório:**
 ````bash
 </> Bash
 git clone [https://github.com/SEU-USUARIO/client-manager-frontend.git](https://github.com/SEU-USUARIO/client-manager-frontend.git)
 ````
2. **Entre na pasta do projeto:**
```bash
</> bash
cd client-manager-frontend
```
3. **Instale as dependências:**
```bash
</> bash
npm install
````
4. **Inicie o servidor de desenvolvimento:**
```bash
</ >bash
npm run dev
```
**O frontend estará disponível em:** http://localhost:5173
